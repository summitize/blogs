# Blogging Site

A simple static blogging site featuring articles about ChatGPT applications.

## Pages

- **Landing Page**: `index.html` - Overview of the blog with links to articles.
- **Blog 1**: `blog1.html` - ChatGPT - An Astrology Champ
- **Blog 2**: `blog2.html` - ChatGPT - An Interview Buddy
- **Blog 3**: `blog3.html` - ChatGPT - Resume Builder

## Running the Site

To view the site locally, you can use a simple HTTP server.

### Using Python (if installed)

Run the following command in the project directory:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000` in your browser.

### Using Node.js (if installed)

Install `http-server` globally:

```bash
npm install -g http-server
```

Then run:

```bash
http-server
```

Open `http://localhost:8080` in your browser.

## Troubleshooting

- Ensure you have a web browser installed.
- If the server doesn't start, check if the port is already in use.
- For any issues, verify file paths and permissions.
